.. module:: matlanticgrrm.log

Logファイルを書くためのモジュール

計算時のLogファイルの中身

+----------+--------+----------------------------------------------+
| LEVEL    | status |                                              |
+==========+========+==============================================+
| DEBUG    |  0     | 完了(収束)                                   |
+----------+--------+----------------------------------------------+
| INFO     |  1     | indexesに指定されていない                    |
|          +--------+----------------------------------------------+
|          |  2     | objectが存在しない(atoms_listの要素がNone)   |
+----------+--------+----------------------------------------------+
| WARNING  |  3     | numpy.linalg.LinAlgError                     |
|          +--------+----------------------------------------------+
|          |  4     | 未収束                                       |
|          +--------+----------------------------------------------+
|          |  5     | タイムアウトで終了                           |
|          +--------+----------------------------------------------+
|          |  6     | forceがmax_fmaxを上回っているため計算を諦めた|
+----------+--------+----------------------------------------------+
| ERROR    |  7     | forceを計算できなかったため計算を諦めた      |
+----------+--------+----------------------------------------------+
| CRITICAL |  8     | 予期しないエラーが発生した                   |
+----------+--------+----------------------------------------------+
