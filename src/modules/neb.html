make_images()

class matlanticgrrm.neb.make_images(eq, connections, nimages, ts=None, same_react=False, same_connect=False, log='nebmake.log')[ソース]

NEBのイメージを作成する.

Parameters:

eq: list of Atoms

EQのAtomsリスト,計算したくないものは要素にNoneを設定しておく.

connections: list of tuple of int

CONNECTION情報.??やDCが含まれていてもOK.

nimages: integer

作成するimageの数

ts: list of Atoms
TS構造のAtomsリスト, imagesの中間に挿入する
現在作成中で使用不可
same_react: bool

Trueにした場合,同じ反応のNEBimageも作成.

same_connect: bool

Trueにした場合,同じeqを繋ぐNEBimageも作成.

log: str or None

logのファイル名. Noneの場合ファイルを出力しない

戻り値

ase.neb.NEB.imagesを要素とするリスト,作成しなかったimageはNone."-"で標準出力

注釈

logファイルを出力する.

ListNEBまで作成するには nebmake() を使用する
